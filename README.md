# CodeAlphaInternshipTasks

Task 1 : Web Scraping
This project extracts book data (Title, Price, Rating) from a web source using Python, BeautifulSoup, and Pandas. This dataset will be used for further analysis in Task 2.

Task 2 : This project performs Exploratory Data Analysis on a book dataset. It includes data cleaning (converting currency strings to floats), statistical profiling, and data visualization using Seaborn and Matplotlib to uncover pricing trends and rating distributions.
